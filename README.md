# Agent Tools

一套功能完整的 Agent Tool System，参考 opencode、claude-code 等热门项目实现。

## 功能列表

### 文件系统
- [ ] Read - 读取文件内容
- [ ] Write - 写入文件内容
- [ ] Edit - 编辑文件内容
- [ ] Glob - 匹配文件路径
- [ ] Grep - 搜索文件内容

### 系统执行
- [ ] Bash - 执行系统命令

### 信息获取
- [ ] WebFetch - 获取网页内容
- [ ] WebSearch - 搜索网页

### 用户交互
- [ ] AskUserQuestion - 向用户提问

### 任务管理
- [ ] TodoWrite / TodoRead - 待办事项读写
- [ ] TaskCreate - 创建任务
- [ ] TaskList - 列出任务
- [ ] TaskUpdate - 更新任务
- [ ] TaskStop - 停止任务
- [ ] TaskOutput - 任务输出

### 规划系统
- [ ] EnterPlanMode - 进入规划模式
- [ ] ExitPlanMode - 退出规划模式

### 高级 Agent 能力
- [ ] NotebookEdit - 笔记本编辑
- [ ] Agent - 子代理
- [ ] EnterWorktree - 进入工作树
- [ ] Team collaboration - 团队协作
- [ ] ToolSearch - 工具搜索
